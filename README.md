# AppealCraft

### The Personal Dispute Advocate

AppealCraft helps everyday consumers fight corporate friction.

Users can describe a problem in plain language, upload supporting evidence, and receive professionally drafted dispute letters with actionable next steps.

Current status: 🚧 Early MVP Development

---

## Problem

Consumers often face:

- Unfair fees
- Denied refunds
- Unreturned security deposits
- Confusing support processes
- Time-consuming escalation paths

Most people know they were treated unfairly.

Very few know how to fight back effectively.

---

## Vision

Transform messy, emotional complaints into:

✅ Professional dispute letters  
✅ Structured legal-style arguments  
✅ Escalation plans  
✅ Clear next actions

---

## Planned MVP Features

### User Input

- Plain text complaint input
- Evidence upload (PDF/images)
- Company/target selection

### AI Processing

- Extract text from uploaded evidence
- Analyze issue context
- Generate structured output

### Output

- Drafted demand letter
- Escalation checklist
- PDF export

## Notes

This project is being built as a Phase 1 MVP focused on validating whether AI can become a practical consumer advocate.
